# Date range 的开始时间的值

```php
再次重声：渲染用 content.     取值用node.

开始时间
{% if content.field_date_from_to %}  
  {{ node.field_date_from_to.value|date('d F Y') }}  
{% endif %}
F表示英文的月份，如：31 July 2023
小d是日期1~31，大D是星期的缩写，1~7.

结束时间
{{ node.field_date_from_to.end_value|date('d F Y') }}

```
